# ChatGPT Branch Map

一个面向 `chatgpt.com` / `chat.openai.com` 的 Chrome 扩展，用来把长对话变成可回溯、可跳转、可分支的结构化导航。

## 核心能力

### 1) 双模式界面
- 轻量模式（默认）：小眼睛 + 问题栏，快速定位问答。
- 高级模式：分支面板（脑图/树形），适合重组和深度回看。

### 2) 节点与分支
- 鼠标移到消息附近可点击 `设为节点`（已支持延迟隐藏）。
- 已设节点支持 `跳到消息`、`复制继续`。
- 支持“建树”自动生成基础结构。
- 支持拖拽节点改父子关系（分支重组）。

### 3) 导航与回溯
- 顶部/底部/上一条/下一条快速跳转。
- 导航列表按“你说（主行）+ ChatGPT（次行）”成组显示。
- 点击导航项跳转到消息**开头**位置（非居中）。
- 支持从子会话返回父会话。

### 4) 拖拽与停靠
- 轻量模式小眼睛支持拖拽 + 自动吸边。
- 高级面板支持标题拖拽 + 自动吸边。
- 双击高级面板标题可复位到右上角。
- 极简/收起面板也可拖拽。

### 5) 其他
- 自动刷新开关（监听新消息变化）。
- 面板尺寸可拖拽调整。
- Mini Map / 树形视图切换。
- 会话级持久化存储（不同会话互不影响）。

## 安装方式

1. 打开 `chrome://extensions/`
2. 打开右上角“开发者模式”
3. 点击“加载已解压的扩展程序”
4. 选择本项目目录（包含 `manifest.json`）
5. 打开 ChatGPT 对话页并刷新

## 使用说明

### 轻量模式（默认）
1. 点击小眼睛展开问题栏。
2. 点 `扫描` 更新当前会话消息。
3. 点击列表中的“你说 / ChatGPT”条目跳转到对应消息开头。
4. 点 `高级` 进入分支面板。

### 高级模式
1. 鼠标移到消息旁点击 `设为节点`。
2. 通过 `树形/脑图` 切换浏览方式。
3. 使用导航区快速定位消息。
4. 拖拽标题可移动面板；双击标题复位到右上角。
5. 点 `轻量` 回到轻量模式。

## 已知限制

- 本扩展不能修改 ChatGPT 服务端真实上下文，只提供可视化导航与回溯辅助。
- ChatGPT 页面结构变化时，可能需要更新选择器逻辑。
- “新聊天中的分支”依赖页面原生入口可见性。

## 项目文件

- `manifest.json`：扩展配置（MV3）
- `content.js`：核心逻辑（扫描、渲染、跳转、拖拽、分支）
- `content.css`：样式
- `lib/interact.min.js`：拖拽库（interact.js）
